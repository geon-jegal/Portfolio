Data(2023-05-23 12:00)

  **Data에 대한 내용을 기술한 readme파일입니다.**

PlayerInfo, 
- 플레이어의 기본 정보를 가지는 데이터

PlayerAttack,   
- 전투와 관련된 기본 정보를 가지는 데이터
- 

PlayerUI,   
- 만약 플레이어가 UI를 이동시키거나, 세부 창을 열었다면 그것에 관한 정보를 담는 데이터
- 하지만 구현을 하다 보니 필요없을 것 같기도 하다.


PlayerPosition,   
- 게임을 키고, 끌때 그리고 맵이동을 할 때 저장 ,사용한다.
- 플레이어가 있는 맵과 상세한 위치(vector값 사용)을 기록한다.


PlayerInventory,   
- 플레이어가 가진 아이템에 관한 정보를 가지는 데이터


PlayerSkill,   
- 플레이어가 사용할 수 있는 스킬, 사용할 수 없는 스킬, 각각의 스킬들의 레벨을 가지는 데이터
- 

PlaterEq,   
- 현재 플레이어가 장착한 장비를 가지는 데이터
- 캐시 장비또한 이곳에 기록할 지는 미지수
- 

PlayerKeySetting,   
- 플레이어가 커스텀 한 키새팅에 관한 정보를 가지는 데이터
- 

PlayerQuest,   
- 플레이어가 완료한 퀘스트, 시작 가능 퀘스트, 진행 중인 퀘스트를 가지는 데이터
